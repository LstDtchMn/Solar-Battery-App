# Runtime dependencies for the KiloVault HLX+ Monitor.
#
# The dashboard, simulator, logging and alarms run on the Python standard
# library alone. The two packages below are only needed for real hardware:
#

#
# Install everything:   pip install -r requirements.txt
# Or just the core (no hardware):   no install needed, use `--simulate`.
